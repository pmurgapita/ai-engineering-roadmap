## ¿Qué he aprendido hoy?
- La regresión lineal se usa para predecir valores numéricos continuos. 
- La regresión predice un número mientras que la clasificación predice una categoría.
- Lineal significa que el modelo asume que la salida puede aproximarse mediante una combinación lineal de las entradas. Para una sola feature: y_pred = wx + b. Geométricamente esto es una recta.
- En la vida real se necesitan varias features. y_pred = x1w1 + x2w2+ ... + b
- Cada peso indica como influye una feature en la predicción. Los pesos pueden ser:
    - Positivos: aumentan la predicción
    - Negativos: reducen la predicción
    - Cercanos a cero: poca influencia
- La regresión lineal aprende los pesos y el sesgo que minimizan la pérdida.
- Si el modelo predice mal el MSE sube y si el modelo predice bien baja.
- El límite es que la regresión lineal solo aprende relaciones aproximadamente lineales. 
- No puedo interpretar la importancia de un feature solo por su peso, debería mirar también la escala de la feature en sí. 

- La regresión lineal es un modelo que aprende una relación lineal entre entradas y una salida numérica ajustando pesos y sesgo para minimizar la pérdida.
- La regresión lineal aprende pesos y sesgo que minimizan la diferencia entre predicciones y valores reales.
