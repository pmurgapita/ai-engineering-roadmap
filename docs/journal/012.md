# Sesión 012

## ¿Qué he aprendido hoy?
- Una neurona lineal completa es y_pred = x * w + b (x por el peso más el sesgo)
- El sesgo permite desplazar la decisión hacia arriba y hacia abajo
- Ahora se necesitan dos gradientes, uno para cada parámetro w y b. El gradiente de w indica cómo ajustar el peso y el gradiente de b indica cómo ajustar el sesgo. 
- Los parámetros w y b no se ajustan de forma limpia. 
- El sesgo hace que el modelo sea más flexible.
- El sesgo permite que una neurona aprenda relaciones que no pasan por cero.
- El peso cambia la inclinación de la recta mientras que el sesgo desplaza la recta.
