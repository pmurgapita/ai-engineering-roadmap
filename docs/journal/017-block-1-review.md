aprender = ajustar parámetros para reducir pérdida y generalizar mejor

Un modelo de Machine Learning aprende cuando ajusta sus parámetros usando datos de entrenamiento para reducir la pérdida y mejorar su rendimiento en datos nuevos.

## Que entiendo ahora
Creo que ahora entiendo cual es el proceso completo del Machine Learning para saber como aprende la inteligencia artificial y como generaliza para dar una respuesta lo más cercana posible a la realidad. 

Un modelo recibe datos representados numéricamente, usa features para hacer predicciones con parámetros, mide el error con una función de pérdida, usa gradientes para ajustar pesos y sesgo, y se evalúa en datos nuevos para comprobar si generaliza.
