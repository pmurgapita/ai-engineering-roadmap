## ¿Qué he aprendido hoy?
- Un tensor es una estructura numérica con una o más dimensiones. El número de dimensiones también se puede llamar rango del tensor. El shape dice el tamaño en cada dimensión. El vector y la matriz también son tensores de una y dos dimensiones. 
- Frase tokenizada ["Me", "gusta", "la", "IA"]
- En LLMs, una forma común es: (batch_size, sequence_length, embedding_dim)
- Una red neuronal puede verse como una cadena de transformaciones de tensores: tensor → transformación → tensor → transformación → tensor
- Cuando entiendes el shape, entiendes que está pasando con los datos. 
- El significado depende del contexto, no solo del shape. El shape describe la estructura, pero el contexto da el significado.
- El número de dimensiones es cuantos números hay en el shape.
- Podemos calcular cuántos números contiene un tensor multiplicando las dimensiones del shape.
- Un tensor es útil en IA porque permite representar datos complejos con múltiples ejes, como batches de imágenes, secuencias de texto, embeddings, activaciones y gradientes.
