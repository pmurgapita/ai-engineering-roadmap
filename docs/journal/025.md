# Sesión 025

## ¿Qué he aprendido hoy?
- Si utilizamos varias capas lineales (componemos funciones), el resultado sigue siendo lineal, seguimos obteniendo una recta. 
- Entonces la red neuronal sigue sin poder aprender relaciones no lineales complejas. Por eso necesitamos meter entre capas una función no lineal: función de activación.
- linear → activation → linear → activation → linear
- Una función de activadcción transforma la salida de una neurona antes de pasarla a la siguente capa, introduce no linealidad.
- La idea venia de algo asi como si la señal es suficientemente fuerte la neurona se activa.
- ReLU: rectified Linear Unit. ReLu(x) = max(0, x) => si x negativo devuelve 0, sino x. Es no lineal porque tiene un codo en cero. 
- Sigmoid: muy negativa cerca de 0, 0 salida 0.5 y muy positiva cerca de 1. Comprime valores entre 0 y 1 Históricamente se usó para probablidades binarias. 
- ReLU introduce no linealidad de forma simple. Sigmoid convierte números en algo parecido a probabilidad.
- La activación es lo que permite que la segunda capa trabaje sobre una representación transformada no linealmente.
