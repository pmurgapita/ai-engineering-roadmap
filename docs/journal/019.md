## ¿Qué he aprendido hoy?
- Matriz como dataset. Conjunto de vectores, cada vector sería un ejemplo.
- Shape es el tamaño de una matriz (filas, columnas). Filas es número de ejemplos, columnas es número de features por ejemplo. (n_examples x n_features)
- Producto matriz-vector. Multiplicamos el vector de pesos por cada fila de la matriz de ejemplos. 
- Batch de ejemplos → predicciones en paralelo. Un batch es un número de ejemplos procesados juntos
- Una capa lineal puede escribirse como: Y = XW + b
X = matriz de entradas
W = matriz de pesos
b = vector de sesgos
Y = matriz de salidas
- La regla de compatibilidad es: (n_examples, input_features) · (input_features, output_features)
= (n_examples, output_features)
