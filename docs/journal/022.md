## ¿Qué he aprendido hoy?
- Una función transforma una entrada en una salida. En Machine Learning un modelo puede verse como una función.
- En Machhine Learning normalmente no conocemos la función correcta, queremos aprenderla. Por eso usamos funciones con parámetros. 
- Cuando entrenamos un modelo estamos intentando encontrar buenos valores para los parámetros. El modelo aprende una función aproximada.
- En ML muchas veces no se aprende la función exacta, sino una aproximación útil.
- Dominio es el conjunto de entrads posibles. 
- Codominio es el tipo de salidas posibles. 
- Podemos encadenar funciones.
- Una red neuronal es una composición de muchas funciones: entrada → capa lineal → activación → capa lineal → activación → salida
- Queremos ajustar los parámetros del modelo para que la función de pérdida sea lo más baja posible
- Un modelo es una función parametrizada; entrenar es ajustar sus parámetros para reducir la pérdida.
