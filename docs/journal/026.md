## ¿Qué he aprendido hoy?
- El forward pass es el proceso de calcular la salida de una red neuronal a partir de una entrada.
- Una neurona oculta no tiene necesariamente un significado humano directo.
- Puede aprender a detectar una combinación útil de features.
- La capa oculta transforma las features originales en nuevas características aprendidas.
- Deep learning aprende sus propias features.
- Usamos varias neuronas para que cada una aprenda un patrón diferente, y los combina en la siguiente capa. 
- Forward pass es utilizar pesos actuales y calcular predicción, mientras que entrenamiento es usar forward pass, clacular pérdida, gradientes, actualizar parámetros y repetir.
- Las neuronas se activan o se apagan dependiendo de la entrada.
