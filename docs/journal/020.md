## ¿Qué he aprendido hoy?
- Podemos tener varias salidas, si queremos calcular de varias entradas varias salidas diferentes. 
- Por cada salida que queramos podemos tener un vector de pesos diferente. No es lo mismo el vector de pesos del precio de una vivienda que el precio de demandas.
- Cada salida es una neurona. 
- Multiplicar una matriz de entradas por una matriz de pesos permite producir varias salidas para muchos ejemplos a la vez.
- cada columna representa una neurona/salida dentro de la misma capa lineal. Una capa lineal con 5 salidas puede verse como 5 neuronas lineales juntas.
- Una capa lineal transforma un vector de entrada en un vector de salida usando una matriz de pesos y un vector de sesgos.
- Una matriz de pesos puede verse como varias neuronas lineales trabajando en paralelo.
