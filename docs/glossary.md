# Feature

Es una característica de algo que nos ayuda a predecir mejor.

---

# Weight

Es el peso, la importancia que tiene dicha características para acercarnos lo más posible con la predicción a la realidad.

---

# bias

Es el sesgo, la base que no cambia aunque cambien las caracteristicas, ayuda a desplazar la función del 0.

---

# loss function

La función de pérdida es aquella que nos ayuda a reconocer cual es la distancia entre las predicciones que el modelo va a dar y la realidad.

---

# Gradiente

Es la inclinación de subida de pérdida. En negativo nos ayuda a descender la pérdida.

---

# Learning rate

Es la cantidad de cambio que vamos metiendo después de cada predicción para ir ajustando el nuevo peso.

---

# Normalización

Es la forma de escalar los datos recibidos en las features para que se puedan utilizar, porque no cambia su influencia sin desestabilizar los gradientes.

---

# Train set

Son los datos utilizados para entrenar el modelo.

---

# Test set

Son los datos reservados para evaluar el modelo ya entrenado, datos que no han sido utilizados para entrenar. 

---

# Generalización

Es el proceso por el cual el modelo ha aprendido un patrón que le permite predecir lo más cercano a la respuesta real.

---

# Overfitting

Es cuando en vez de generalizar se aprende demasiado restringido a los ejemplos, con un train loss muy bajo y un test loss alto. 

--- 

# Underfitting

Es cuando con los datos no se genera un buen aprendizaje ni si quiera a nivel train únicamente. Tanto el train loss como el test loss es alto.
